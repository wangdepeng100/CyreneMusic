# 配置洛雪音乐第三方音源

本指南说明如何在 Cyrene Music 中配置洛雪音乐第三方音源。

## 前提条件

1. 拥有一个可用的洛雪音源服务器 API 地址
2. 如果服务器需要验证，需要知道 API Key

## 配置步骤

### 1. 进入音源设置

- **Windows (Fluent UI)**: 设置 → 音源设置
- **Android/iOS**: 设置 → 音源设置

### 2. 选择音源类型

在"音源类型"下拉菜单中选择 **"洛雪音乐"**。

### 3. 配置音源地址

输入洛雪音源服务器的 API 地址，例如：
- `https://lxmusicapi.example.com`
- `http://your-server:port`

### 4. 配置 API Key（可选）

如果音源服务器需要验证密钥，在"API Key"输入框中输入验证密钥。

例如：`share-v2`

### 5. 测试连接

点击"测试连接"按钮验证配置是否正确。

### 6. 保存配置

点击"保存配置"按钮保存设置。

## 支持的音乐平台

洛雪音源支持以下音乐平台：

| 平台 | 来源代码 | 支持状态 |
|------|----------|----------|
| 网易云音乐 | wy | ✅ 支持 |
| QQ音乐 | tx | ✅ 支持 |
| 酷狗音乐 | kg | ✅ 支持 |
| 酷我音乐 | kw | ✅ 支持 |
| Apple Music | - | ❌ 不支持（请使用 OmniParse） |

## 音质支持

洛雪音源支持的音质等级：
- 标准音质 (128k)
- 极高音质 (320k)
- 无损音质 (flac)
- Hi-Res (flac24bit)

## 注意事项

1. 洛雪音源只返回音频 URL，不提供歌词。歌词将从后端服务获取。
2. 如果某个平台的歌曲无法播放，可能是该音源服务器不支持该平台或该歌曲。
3. 请确保音源服务器稳定可用。

## 常见错误

| 错误代码 | 含义 | 解决方案 |
|----------|------|----------|
| code: 1 | IP 被封禁 | 稍后重试或更换网络 |
| code: 2 | 获取链接失败 | 检查歌曲是否可用 |
| code: 4 | 服务器内部错误 | 检查音源服务器状态 |
| code: 5 | 请求过于频繁 | 降低请求频率 |
| code: 6 | 参数错误 | 检查配置是否正确 |
